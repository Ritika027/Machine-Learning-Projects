# 📊 Real-Time Sports Odds Data (Multiple Bookmakers)

## 🔗 Dataset Link

Kaggle: https://www.kaggle.com/datasets/ritika027/real-time-sports-odds-data-multiple-bookmakers

---

## 📌 Overview

This project contains a real-time sports betting odds dataset collected using an external API. The dataset includes match details along with betting odds from multiple bookmakers.

The data has been processed and transformed into a structured format using Python and Pandas, making it suitable for data analysis, visualization, and machine learning applications.

---

## ⚙️ Features

* Sport category (`sport_key`)
* Match start time (`commence_time`)
* Home team and Away team
* Head-to-head betting odds for both teams
* Spread betting points
* Aggregated odds from multiple bookmakers

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas 📊
* Requests (API handling)
* JSON Data Processing

---

## 🔍 Data Collection Process

1. Data is fetched using a sports odds API.
2. JSON response is received containing nested structures.
3. Relevant features are extracted (match info, odds, spreads).
4. Data is cleaned and converted into a Pandas DataFrame.
5. Final dataset is exported as CSV and uploaded to Kaggle.

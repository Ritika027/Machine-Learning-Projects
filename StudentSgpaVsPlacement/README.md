# 🎓 Placement Prediction using Machine Learning

## 📌 Project Description

This project is a simple Machine Learning model that predicts whether a student will get placed or not based on two important factors:

* **IQ (Intelligence Quotient)**
* **CGPA (Cumulative Grade Point Average)**

The model analyzes the relationship between these features and placement outcomes to make predictions.

---

## 🚀 Objective

The main goal of this project is to:

* Understand basic Machine Learning workflow
* Build a classification model
* Visualize decision boundaries
* Predict placement status based on student data

---

## 🧠 Technologies Used

* Python 🐍
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* mlxtend (for decision boundary visualization)

---

## 📊 Dataset

The dataset contains:

* **IQ** → Intelligence level of student
* **CGPA** → Academic performance
* **Placement** → Output (1 = Placed, 0 = Not Placed)

---

## ⚙️ Working Process

1. Import dataset
2. Data preprocessing
3. Feature selection (IQ & CGPA)
4. Train-test split
5. Model training using classification algorithm
6. Model evaluation
7. Visualization using decision boundary

---

## 🤖 Model Used

A classification algorithm such as:

* Logistic Regression
  *(can be extended to KNN, SVM, etc.)*

---

## 📈 Output

* Predicts whether a student will be placed or not
* Visualizes decision regions for better understanding

---

## 📌 Example

Input:

* IQ = 120
* CGPA = 8.5

Output:

* ✅ Placed / ❌ Not Placed

---

## 🎯 Future Improvements

## 🙌 Conclusion

This project demonstrates a basic Machine Learning classification problem and helps in understanding how student performance metrics can influence placement outcomes.

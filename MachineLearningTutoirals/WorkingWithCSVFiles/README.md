# 📊 CSV File Handling in Python (Jupyter Notebook)

## 📌 Project Overview

This project is a Jupyter Notebook-based implementation where I explored and learned how to work with **CSV (Comma-Separated Values) files in Python**. The notebook demonstrates various operations such as reading, writing, modifying, and analyzing CSV data using built-in Python libraries and commonly used modules.

The goal of this project is to understand how structured data is stored in CSV format and how Python can be used to efficiently manipulate and process such data.

---

## 🎯 Objectives

* Understand the structure and format of CSV files
* Learn how to read and write CSV files in Python
* Perform data manipulation and transformation
* Handle real-world datasets using CSV operations
* Gain hands-on experience with file handling and data processing

---

## 🛠️ Tools & Libraries Used

* Python
* Jupyter Notebook
* `csv` module (built-in)
* `pandas` (for advanced operations)

---

## 📚 Topics Covered

### 1. Reading CSV Files

* Reading data using `csv.reader()`
* Reading CSV files using `pandas.read_csv()`
* Accessing rows and columns

### 2. Writing to CSV Files

* Writing data using `csv.writer()`
* Creating new CSV files
* Appending data to existing files

### 3. Data Manipulation

* Filtering specific rows
* Selecting columns
* Updating values in CSV data

### 4. Working with Pandas

* DataFrame creation from CSV
* Basic operations like head(), tail(), describe()
* Data cleaning and preprocessing

### 5. Handling Real Data

* Managing missing values
* Data formatting and conversion
* Sorting and analyzing CSV data
